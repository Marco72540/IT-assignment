Cloud Gaming also known as game streaming, cloud gaming refers to using cloud servers to run games instead of using local devices.
Basically, cloud gaming does not required users to download the game to the users' devices and
The games will run on the remote servers and stream the games to the users' devices.

Example of Cloud Gaming---Google Stadia

What is Google Stadia

Google Stadia is a very common cloud gaming service, it is operate by one of the biggest company in the world. So its safe and trustable.

Google Stadia is a cloud gaming service and games storefront operated by Google.
What Stadia promises is a game-streaming experience that only requires the most basic of equipment

  1. Chromecast Ultra ,smartphone desktop or laptop
  2. Controller
    • Google’s own Stadia Controller
    • The Xbox One controller
    • The PS4’s DualShock4 gamepad
  3. Connection to the internet (Google Stadia works on 10Mbps connections and only requires 35Mbps for full 4K HDR/60fps.)

Starting a game is simple. There’s no need for an expensive console or gaming rig. There’s no need to install from a disc or cartridge. There’s nothing to download. Just buy, hit play, and you’re in.

Features of Google Stadia

  • Only requiring the device to have an Internet connection and support for Google Chrome
  • Combine YouTube's function for video game live streams
  • Good connection in anywhere, as the large number of data centers across the globe
  • Support the streaming of games in HDR at 60 frames per second with 4K resolution
  • Players can start games without having to download content to personal device.
  • Self-developed controller which directly connects to the data center that reduce input latency
