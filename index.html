Background information

Nowadays, as the demands of gaming quality is being higher and higher, many gamers are not satisfy to their own devices. Since hardwares are evolving everyday, if you are not rich enough, your gaming experience will be limited. Due to these development, the Cloud Gaming has been invented. To help those gamer which not have ability to purchse a better gaming outfits.


What is Cloud Gaming?

Cloud Gaming also known as game streaming, cloud gaming refers to using cloud servers to run games instead of using local devices.
Basically, cloud gaming does not required user to download the game to the their device.
The games will run on the remote servers and the games will streamed to users' device.


How does cloud gaming work?

The basic idea of cloud gaming is like the other streaming platforms, such as "Netflix", "YouTube". You will play your games on the server which owned by the companies that provide services. Once you choose a game to play, the game will run in the server. Then, every actions you made will become a command and send it to the server. The server will execute the commands and streamed back to you with a small latency.

  1. User made any actions with the countroller.
  2. All the action made will use the internet send to the Game Servers.
  3. The server received the action and input the action into the game.
  4. The game will get the action and refresh the screen.
  5. The user will received the screen after the action input.
  6. The loop will run again and again to allow user to play the game through internet.


Types of Cloud Gaming

Game/Video Streaming

Player controls the game and it executed remotely. Meanwhile, the server processes the operations and render the actual game and image. Then, the video results are transmitted directly to a player’s computer or device.
The server is capable for processing needs so that it allows player to play high resolution game without powerful console or device.

Command Gaming

Command gaming is kind of like game streaming because they are both stored and executed remotely. But the difference between these is that Command gaming is only process game logic and graphic commands while rendering occurs locally on the player’s device.
It could reduce the latency because it does not need to transmit video.It requires less bandwidth from internet.

Progressive Downloading

Player need to download a small number of files to their device before playing the game. It could reduce the cost of sending patch and producing media.
One of the disadvantages is that it does not support cross cross-platform.
